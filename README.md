# RESTFUL API MEANB

## Dependencies

- npm
- node
- ng

## Set Up

- clone this repository
 
 `git clone https://github.com/juampam/digitalLibrary `

- install modules

In each folder (backend/, frontend/). Type:

`npm install`

or

`npm i`

# - Start servers

- Backend

In the backend/ diretory type:

`npm start`

- FrontEnd

in the frontend/ directory type:

`ng start`
